Этап 2
Реализовал ui для мобилки приложения тренировок на flutter
В мобильном приложении есть (псевдо) регистрация и вход, то есть там есть минимальная валидация полей, но данные никак не проверяются, сразу считается что они правильные (реализация на моках)
Далее есть основная активность, где имеются вкладки календаря тренировок, списка тренировок, списка упражнений, списка платной продукции с корзиной и вкладка профиля
Все кнопки и поля поиска рабочие, но данные сохраняются только локально (не реализован бекенд)
